# Cài đặt

- Bước 1: Clone dự án

- Bước 2: Chạy câu lệnh `yarn install`

- Bước 3: Copy file .env.example -> .env.development.local

- Bước 4: Chạy câu lệnh `yarn dev`

# Công nghệ

1. Vite

2. Jest

3. Tailwindcss

4. i18n

5. storybook

6. husky / commitlint

7. redux

8. react-query

9. Những thư viện sử dụng: lodash, classnames, moment, antd (tùy thuộc vào từng dự án)
